# PlantCareAI

PlantCareAI uses transfer learning (MobileNetV2) to classify plant leaf diseases and provide care recommendations.

## Prerequisites (what the tracker checks)
- `requirements.txt` or `environment.yml` must be present in the repository root.
- `README.md` must be present in the repository root.
